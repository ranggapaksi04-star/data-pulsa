README — data-pulsa

Pulsa Transaction History Tracker

Python CLI program to record and manage mobile credit (pulsa) transactions.

Problem

My family runs a small pulsa shop that records every transaction manually in a notebook. This program was built to digitize that process.

Features


Add transaction
Display history
Search by phone number
Calculate total spending
Exit


How to Run

python data_pulsa.py

Tech


Python 3
VSCode


Known Limitation

No data persistence — all data resets when the program closes. File-based storage planned for next version.
